# TradeNote 프로젝트 개요

## 📌 프로젝트 소개

**TradeNote**는 개인 투자자를 위한 통합 주식 거래 및 분석 플랫폼입니다. 
AI 기반 차트 분석, 실시간 시장 데이터, 매매일지 관리 등 주식 투자에 필요한 모든 기능을 
한 곳에서 제공하는 웹 애플리케이션입니다.

---

## 🎯 프로젝트 목표

1. **투자 습관 개선**: 체계적인 매매일지 관리를 통한 투자 패턴 분석
2. **데이터 기반 의사결정**: AI 분석과 실시간 시장 데이터를 활용한 투자 판단 지원
3. **사용자 경험 최적화**: 직관적이고 간편한 UI/UX로 초보자도 쉽게 사용 가능
4. **실시간 정보 제공**: 차트, 뉴스, 호가 등 실시간 시장 정보 통합 제공

---

## 🚀 주요 기능

### 1. 대시보드 (Dashboard)
- **실시간 차트 분석**
  - TradingView Lightweight Charts 기반 고성능 차트
  - 다양한 시간 간격 지원 (5분, 10분, 15분, 30분, 60분, 120분, 240분)
  - 일/주/월/년 단위 기간 선택
  - 동적 지지/저항선, 이동평균선 자동 표시

- **AI 차트 분석**
  - 기술적 지표 분석 (RSI, MACD, 볼린저 밴드 등)
  - 매매 신호 생성 (BUY/SELL/HOLD)
  - 지지/저항선 자동 감지
  - 차트 패턴 인식
  - 리스크 분석

- **호가창 (Order Book)**
  - 실시간 매수/매도 호가 표시
  - 한국/미국 주식 지원

- **즐겨찾기**
  - 종목별 즐겨찾기 관리
  - 사용자별 데이터베이스 저장

- **종목별 뉴스**
  - 선택한 종목 관련 뉴스 자동 필터링
  - 실시간 뉴스 피드

### 2. 매매일지 (Trading Journal)
- **매매 내역 기록**
  - 일별/월별 매매 내역 관리
  - 수익/손실 자동 계산
  - 차트 기반 수익률 시각화

- **통계 분석**
  - 총 수익금 집계
  - 기간별 수익률 분석
  - 매매 패턴 분석

### 3. AI 분석 (AI Analysis)
- **차트 이미지 분석**
  - 차트 이미지 업로드 시 AI 분석
  - PVG (Price Volume Gap) 감지
  - 상승/하락 라인 분석
  - 투자 추천 생성

- **AI 챗봇**
  - 주식 시장 관련 질의응답
  - 실시간 뉴스 및 시장 데이터 기반 답변
  - Ollama 기반 LLM 활용

### 4. 뉴스 (News)
- **실시간 뉴스 피드**
  - 한국/미국 경제 뉴스 분리 제공
  - RSS 피드 기반 뉴스 수집
  - AI 기반 뉴스 요약
  - 종목별 뉴스 필터링

### 5. 커뮤니티 (Community)
- **투자자 커뮤니티**
  - 실시간 반응 확인
  - 투자 인사이트 공유

### 6. 캘린더 (Calendar)
- **일정 관리**
  - 중요한 시장 이벤트 관리
  - 매매 일정 추적

### 7. 사용자 인증 및 관리
- **회원 관리**
  - 회원가입/로그인
  - 프로필 관리
  - 비밀번호 변경
  - 계정 삭제

## 🛠 기술 스택

### Frontend
- **HTML5 / CSS3 / JavaScript (Vanilla)**
  - 프레임워크 없이 순수 JavaScript로 구현
  - 반응형 웹 디자인
  - 다크/라이트 모드 지원

- **차트 라이브러리**
  - TradingView Lightweight Charts
  - 고성능 실시간 차트 렌더링

### Backend
- **Python FastAPI** (포트 8000)
  - AI 분석 서비스
  - 시장 데이터 API
  - 뉴스 수집 및 요약
  - 차트 분석 API

- **Java Spring Boot** (포트 8001)
  - 사용자 인증 및 관리
  - 매매일지 CRUD
  - 즐겨찾기 관리
  - RESTful API 제공

### Database
- **MySQL**
  - 사용자 정보 저장
  - 매매일지 데이터
  - 즐겨찾기 데이터

### AI/ML
- **Ollama**
  - LLM 기반 챗봇
  - 차트 이미지 분석 (Vision Model)

- **Transformers (Hugging Face)**
  - 뉴스 헤드라인 요약
  - 한국어 번역

### 외부 API 통합
- **yfinance**: 미국 주식 데이터
- **FinanceDataReader**: 한국 주식 데이터
- **Finnhub**: 시장 데이터 (선택적)
- **RSS Feeds**: 뉴스 수집
