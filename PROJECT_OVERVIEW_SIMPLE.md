# 프로젝트 개요

## 문제점

### 1. 주식 투자 정보가 여러 곳에 흩어져 있고, 체계적으로 관리하기 어려움
- 차트, 호가, 뉴스 등 투자에 필요한 정보가 각각 다른 사이트에 분산되어 있음
- 매매 내역을 체계적으로 기록하고 분석할 방법이 없음
- 관심 종목의 정보를 한 곳에서 확인하기 어려움

### 2. 초보 투자자가 차트를 분석하고 투자 판단을 내리기 어려움
- 기술적 지표를 이해하고 활용하기 어려움
- 차트 패턴과 지지/저항선을 파악하기 어려움
- 객관적인 매매 신호를 얻기 어려움

---

## 해결책 (주요 기능)

### 1. 통합 대시보드
**실시간 차트, 호가, 뉴스를 한눈에 확인**
- TradingView 기반 고성능 실시간 차트
- 다양한 시간 간격 지원 (5분~240분, 일/주/월/년)
- 실시간 호가창 (매수/매도 호가)
- 종목별 관련 뉴스 자동 필터링
- 현재 위치 기반 종목 데이터를 한눈에 확인

### 2. AI 차트 분석
**자동 기술적 지표 분석 및 매매 신호 생성**
- 기술적 지표 자동 계산 (RSI, MACD, 볼린저 밴드 등)
- 매매 신호 자동 생성 (BUY/SELL/HOLD)
- 지지/저항선 자동 감지 및 표시
- 차트 패턴 인식 및 리스크 분석
- 차트 이미지 업로드 시 AI 분석 (PVG, 상승/하락 라인)

### 3. 매매일지 관리
**체계적인 매매 내역 기록 및 분석**
- 일별/월별 매매 내역 관리
- 수익/손실 자동 계산
- 차트 기반 수익률 시각화
- 총 수익금 및 기간별 통계 분석
- 매매 패턴 분석을 통한 투자 습관 개선

### 4. 종목별 맞춤 정보
**관심 종목의 정보를 한 곳에서 관리**
- 즐겨찾기 기능으로 관심 종목 저장
- 종목별 관련 뉴스 자동 필터링
- 실시간 가격 정보 및 변동률 표시
- AI 챗봇을 통한 종목 관련 질의응답

---

## 기술 스택

- **Frontend**: HTML/CSS/JavaScript, TradingView Lightweight Charts
- **Backend**: Python FastAPI (AI 분석), Java Spring Boot (인증/데이터)
- **Database**: MySQL
- **AI/ML**: Ollama (LLM), Transformers (뉴스 요약)
- **외부 API**: yfinance, FinanceDataReader, RSS Feeds

---

## 기대 효과

✅ **투자 효율성 향상**: 모든 투자 정보를 한 곳에서 관리  
✅ **의사결정 지원**: AI 분석을 통한 객관적 투자 판단  
✅ **습관 개선**: 체계적인 매매일지 관리를 통한 투자 패턴 분석  
✅ **접근성 향상**: 초보자도 쉽게 사용할 수 있는 직관적 인터페이스
