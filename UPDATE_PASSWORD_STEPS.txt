RDS 비밀번호 재설정 단계
=======================

1. AWS 콘솔 접속
   https://ap-southeast-2.console.aws.amazon.com/rds/

2. 데이터베이스 선택
   - RDS → 데이터베이스 → share-db

3. 수정 클릭
   - 오른쪽 상단 "수정" 버튼

4. 비밀번호 변경
   - "암호" 섹션으로 스크롤
   - "새 마스터 암호" 입력 (예: NewPassword123!@#)
   - "암호 확인"에 동일 입력

5. 적용 방식 선택
   - "수정 사항 즉시 적용" (빠름, 다운타임 있음)
   - 또는 "유지 관리 기간 동안 적용" (느림, 다운타임 없음)

6. 변경 완료
   - "계속" → "수정 사항 확인" → "수정"

7. 완료 대기
   - 즉시 적용: 몇 분 소요
   - 유지 관리: 다음 유지 관리 시간

8. application.yml 업데이트
   - 새 비밀번호 입력
   - JAR 재빌드 및 배포

새 비밀번호를 알려주시면 application.yml을 업데이트하고 배포하겠습니다!
