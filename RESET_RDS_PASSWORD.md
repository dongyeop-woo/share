# RDS 비밀번호 재설정 가이드

## 문제
RDS 마스터 비밀번호를 모르는 상황입니다.

## 해결 방법

### 방법 1: AWS 콘솔에서 비밀번호 재설정 (권장)

**1단계: RDS 콘솔 접속**
- AWS 콘솔 → RDS → 데이터베이스 → `share-db` 선택

**2단계: 비밀번호 변경**
1. "수정" 버튼 클릭
2. "암호" 섹션으로 스크롤
3. "새 마스터 암호" 입력:
   - 새 비밀번호 입력 (예: `NewPassword123!@#`)
   - "암호 확인"에 동일하게 입력
4. 페이지 맨 아래로 스크롤
5. "수정 사항 즉시 적용" 또는 "유지 관리 기간 동안 적용" 선택
   - **주의:** 즉시 적용하면 데이터베이스가 재시작됩니다
6. "계속" → "수정 사항 확인" → "수정" 클릭

**3단계: 비밀번호 변경 완료 대기**
- "즉시 적용" 선택했다면 몇 분 소요
- "유지 관리 기간" 선택했다면 다음 유지 관리 시간에 적용

**4단계: application.yml 업데이트**
- 새 비밀번호로 `application.yml` 수정
- 새 JAR 빌드 및 배포

### 방법 2: AWS CLI로 비밀번호 재설정

```bash
aws rds modify-db-instance \
    --db-instance-identifier share-db \
    --master-user-password "NewPassword123!@#" \
    --apply-immediately
```

### 방법 3: 다른 파일에서 비밀번호 찾기

다음 위치에서 비밀번호를 찾아보세요:
- 다른 설정 파일
- 환경 변수
- 비밀번호 관리자
- 이전에 작성한 문서

## 새 비밀번호 설정 후

1. `application.yml` 업데이트:
```yaml
spring:
  datasource:
    username: admin
    password: 새비밀번호
```

2. 새 JAR 빌드 및 배포
3. Spring Boot 재시작

## 중요 사항

- **비밀번호는 재확인 불가능**: AWS에서는 기존 비밀번호를 볼 수 없습니다
- **재설정 필요**: 비밀번호를 재설정해야 합니다
- **다운타임**: 즉시 적용 시 짧은 다운타임 발생 가능
- **연결 정보**: 새 비밀번호는 모든 연결(Spring Boot, MySQL Workbench 등)에 적용됩니다
